Proyecto de investigación para la materia de Física

///  OBJETIVOS  ///
El objetivo de este proyecto es elaborar un reporte de investigación basado en un caso práctico que involucra el uso de ondas de luz
para la obtención de una magnitud física. En este caso, se busca medir el ritmo cardíaco de una persona y representar gráficamente los resultados obtenidos.
Asimismo, el proyecto contempla el diseño y la construcción del sensor necesario, así como el desarrollo de un prototipo funcional
que permita llevar a cabo dichas mediciones.
